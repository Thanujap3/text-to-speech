About This API

This is a complete Text-to-Speech (TTS) system with Natural Language Processing (NLP) capabilities.

Features:

  • ✓ Advanced text preprocessing
  • ✓ Tokenization and cleaning
  • ✓ Real-time speech synthesis
  • ✓ Adjustable speed and volume
  • ✓ REST API endpoints

API Endpoints:

POST /api/synthesize - Full synthesis
POST /api/preprocess - Preprocessing only
POST /api/pipeline - Complete pipeline
GET /api/health - Health check
GET /api/info - API info

Version: 1.0.0 | Built with Flask + pyttsx3 | 2026
